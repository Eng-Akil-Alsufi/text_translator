# Text Translator

## üåê Project Overview

The **Text Translator** is a modern, responsive web application designed for instant and seamless text translation between multiple languages. Built with a focus on user experience, it provides a clean interface for quick translation tasks.

The application is styled using **Bootstrap 4** and a custom CSS theme, ensuring a professional and visually appealing design that is fully responsive across different devices.

## ‚ú® Key Features

*   **Multi-Language Support:** Allows translation between a wide range of popular languages (English, Arabic, French, German, Spanish, Italian, Russian, Chinese, Japanese).
*   **Instant Translation:** Provides immediate translation results upon user request.
*   **Text Alignment:** Automatically adjusts text alignment (`dir="auto"`) based on the selected language (Right-to-Left for Arabic).
*   **Utility Functions:** Includes features to copy the translated text to the clipboard and download the translation as a file.
*   **Responsive Design:** Optimized for use on both desktop and mobile devices.
